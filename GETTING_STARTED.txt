ENVIRONMENT VARIABLES
===================

.env.local contains:
- NEXT_PUBLIC_API_MODE: Set to "mock" for testing without backend
- NEXT_PUBLIC_API_URL: Backend API URL (for real mode)
- NEXT_PUBLIC_GOOGLE_MAPS_API_KEY: Dummy key for testing
- FONNTE_API_KEY: WhatsApp API (simulated)

---

MOCK MODE vs REAL API MODE
===========================

MOCK MODE (Default):
- No backend needed
- All features work
- Data stored in localStorage
- Perfect for frontend development
- Set: NEXT_PUBLIC_API_MODE=mock

REAL API MODE:
- Requires Laravel backend
- Data stored in PostgreSQL database
- For production deployment
- See BACKEND_INTEGRATION.md for setup
- Set: NEXT_PUBLIC_API_MODE=real
